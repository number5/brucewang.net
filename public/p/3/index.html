  9. 升级WP-OpenID plugin 到3.0, 改用新的主题 Fauna

    Sat 04 October 2008

    今天将WP-OpenID插件升级到3.0， 现在支持OpenID Server模式了，就是说装了这个插件就可以直接用你的blog url作为openid url。 OpenID Server 模式需要另外安装一个XRDS-Simple插件

    另外，原来Web2.0主题用得烦了，ajax comment和openid 插件的冲突一直也解决不了，干脆就换一个Fauna主题，感觉还不错，细节慢慢调吧。
